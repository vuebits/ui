## Installation

`npm i @vuebits/ui` / `yarn add @vuebits/ui`

And install in your entry file (e.g. `main.js`):

```javascript
import { createUI } from '@vuebits/ui';

createApp(App).use(createUI({ /* your config here */ })).mount('#app');
```
